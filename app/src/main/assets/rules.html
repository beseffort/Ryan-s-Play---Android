*Players*
Guess where the batter will hit the ball on the field before the first pitch of an at bat in a live baseball game. You must select a position on the field or the Strikeout/Walk/Hit By Pitch button for every Play. If no selection is made, then you have no chance to win points for that play.

1 batter = 1 Play

If you think the batter will get a base hit, select the position the hit will go to and then turn on the Base Hit light.

If you think a run will score or the batter will hit a home run during the at bat, select a position and turn on the Score Run light, and/or the Home Run light.

*Umpires*
If you created the game then you are the default umpire. Umpires are in charge of closing the guessing period before the first pitch is thrown to the batter, and entering the result of the play. The result of the play should include the first fielder to have control of the ball after it is hit, not the first fielder to touch the ball.

Umpires are also players in the games they create, so make sure that results entered are fair and accurate.
